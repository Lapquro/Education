# Лабораторная работа: bitsets (для получения навыков работы с битовыми операциями)

## Задание

На вход подаются слова, разделённые произвольными символами в произвольном количестве.  
**Слово** — любая последовательность букв латинского алфавита (верхний или нижний регистр).

**Вариант:** Есть ли слово, все согласные которого являются глухими?

---

## Требования к решению

- Запрещается хранить входные данные в массиве
- Используется `bitset`
- Обрабатываются все входные данные, предусмотренные заданием
- Код читаемый, названия переменных отражают смысл
